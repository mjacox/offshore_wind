# offshore_wind
Code and data to reproduce figures from Jacox et al., "Physical and biogeochemical ocean response to potential offshore wind development along the U.S. west coast"

FILES:
plot_figs_jacox_grl_2026.m: Matlab script containing functions to load in appropriate data and plot each figure
/dependencies: Several scripts needed to run the plotting functions
fig*.mat: Data files for each figure
coastline.mat: A matlab file with coastline coordinates

Place the data files in the matlab working directory, otherwise insert their path within the matlab script
